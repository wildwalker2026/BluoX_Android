# BlueBox AI Android

> Your pocket AI assistant, enjoy AI conversations on the go.

> ⚠️ **Note:** This application currently supports **Simplified Chinese only**.

## ✨ Features

- 🤖 **Multi-Model Support** — OpenAI, DeepSeek, Qwen, Zhipu, Kimi, MiniMax and more
- 💬 **Smooth Chat** — Streaming output for real-time AI responses
- 📝 **Knowledge Base** — Upload documents to build a knowledge base for AI reference
- 🎨 **Multi-Agent** — Create multiple AI roles with independent configurations
- 🖼 **Image Recognition** — Take photos or select images for AI visual understanding
- 📂 **File Management** — AI can browse, read and edit files
- 🔒 **Local First** — All data stored locally, no privacy uploaded
- 🌙 **Dark Mode** — Auto-switch between light/dark themes following system settings

## 📥 Download

Go to [Releases](../../releases/latest) to download the latest APK.

> Supports Android 7.0+

## 🚀 Quick Start

1. Download and install the APK
2. Choose your familiarity with AI APIs on first launch
3. Enter your AI provider's API Key in settings
4. Start chatting with AI

# 小蓝AI盒子 Android

> 随身携带的 AI 智能助手，手机端也能畅享 AI 对话。

## ✨ 功能特性

- 🤖 **多模型支持** — 支持 OpenAI、DeepSeek、通义千问、智谱、Kimi、MiniMax 等主流 AI 服务商
- 💬 **流畅对话** — 支持流式输出，实时查看 AI 回复
- 📝 **知识库** — 上传文档建立知识库，AI 回答时自动检索参考
- 🎨 **多智能体** — 支持创建多个 AI 角色，每个角色独立配置
- 🖼 **图片识别** — 拍照或选择图片发送给 AI，支持视觉理解
- 📂 **文件管理** — AI 帮你浏览、读取、编辑文件
- 🔒 **本地运行** — 所有数据存储在本地，不上传任何隐私信息
- 🌙 **深色模式** — 支持跟随系统自动切换深色/浅色主题

## 📥 下载安装

前往 [Releases](../../releases/latest) 下载最新版 APK。

> 支持 Android 7.0 及以上

## 🚀 快速开始

1. 下载 APK 并安装
2. 首次启动选择你对 AI API 的了解程度
3. 在设置中填入你的 AI 服务商 API Key
4. 开始和 AI 对话
