# BlueBox AI PC

> A local AI assistant that can directly control your computer.

> ⚠️ **Note:** This application currently supports **Simplified Chinese only**.

## ✨ Features

- 🤖 **Multi-Model Support** — OpenAI, DeepSeek, Qwen, Zhipu, Kimi, MiniMax and more
- 📁 **File Management** — AI can browse, read, edit and create files with diff preview and confirmation
- 💻 **Command Execution** — AI can run system commands for automation tasks
- 🌐 **Browser Control** — AI can open web pages, click, type and take screenshots
- 📸 **Screen Capture** — One-click screenshot to let AI see your screen
- 📝 **Knowledge Base** — Upload documents to build a knowledge base for AI reference
- 🗂 **Notebook** — Built-in notebook with automatic operation logging
- 📋 **Plan Mode** — Plan complex tasks step by step before execution
- 🔒 **Local First** — All data stored locally, no privacy uploaded
- 🎨 **Multi-Agent** — Create multiple AI roles with independent configurations

## 📥 Download

Go to [Releases](../../releases/latest) to download the latest installer.

> Supports Windows

## 🚀 Quick Start

1. Download and install `BlueBox Setup.exe`
2. Choose your familiarity with AI APIs on first launch
3. Enter your AI provider's API Key in settings
4. Start chatting with AI to manage your computer

## ⚙️ Requirements

- Windows 10 or later
- Internet connection (for AI API calls)

# 小蓝AI盒子 PC

> 本地运行的 AI 智能助手，让 AI 直接操控你的电脑。

## ✨ 功能特性

- 🤖 **多模型支持** — 支持 OpenAI、DeepSeek、通义千问、智谱、Kimi、MiniMax 等主流 AI 服务商
- 📁 **文件管理** — AI 直接帮你浏览、读取、编辑、创建文件，支持 diff 预览和确认机制
- 💻 **执行命令** — AI 可以执行系统命令，帮你完成自动化任务
- 🌐 **浏览器操控** — AI 可以打开网页、点击、输入、截图
- 📸 **屏幕截图** — 一键截图发送给 AI，让它看懂你的屏幕
- 📝 **知识库** — 上传文档建立知识库，AI 回答时自动检索参考
- 🗂 **笔记本** — 内置笔记本，AI 可自动记录操作日志
- 📋 **计划模式** — 复杂任务先规划再执行，逐步确认
- 🔒 **本地运行** — 所有数据存储在本地，不上传任何隐私信息
- 🎨 **多智能体** — 支持创建多个 AI 角色，每个角色独立配置

## 📥 下载安装

前往 [Releases](../../releases/latest) 下载最新版安装包。

> 支持 Windows 系统

## 🚀 快速开始

1. 下载并安装 `小蓝AI盒子 Setup.exe`
2. 首次启动选择你对 AI API 的了解程度
3. 在设置中填入你的 AI 服务商 API Key
4. 开始和 AI 对话，让它帮你管理电脑

## ⚙️ 系统要求

- Windows 10 及以上
- 网络连接（用于调用 AI API）
